## How to Use This Repo

### Compiling and Running Java Files

```bash
# Navigate to a course folder
cd 02-java-programming-for-beginners

# Compile a Java file
javac -d classes src/DataTypes.java

# Run the compiled class
java -cp classes DataTypes
```

### Prerequisites

- **Java 21+** вЂ” [Download OpenJDK](https://openjdk.org/)
- **Git** вЂ” [Install Git](https://git-scm.com/)
- **IDE** (optional) вЂ” IntelliJ IDEA or VS Code with Java extensions
